# 动态组件

通过使用保留的 `<component>` 元素，动态地绑定到它的`is`特性，我们让多个组件可以使用同一个挂载点，并动态切换：

```html
<component v-bind:is="currentView">
    <!-- 组件在 vm.currentview 变化时改变！ -->
</component>
<script>
var vm = new Vue({
    el: '#app',
    data: {
        currentView: 'home'
    },
    components: {
        home: { /* ... */ },
        posts: { /* ... */ },
        archive: { /* ... */ }
    }
})
</script>
```

## keep-alive

如果把切换出去的组件保留在内存中，可以保留它的状态或避免重新渲染。为此可以添加一个 keep-alive 指令参数：

```html
<keep-alive>
    <component :is="currentView">
        <!-- 非活动组件将被缓存！ -->
    </component>
</keep-alive>
```
